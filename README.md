# Iris_Classification_Assignment
this assignment upload for alfidotech 
Project Title: Iris Species Classification

Dataset:
https://www.kaggle.com/datasets/bhanupratapbiswas/iris-classification-dataset

Objective:
To build a machine learning model to classify iris flower species using sepal and petal measurements.

Algorithms Used:
- k-Nearest Neighbors
- Logistic Regression
- Decision Tree

Evaluation Metrics:
- Accuracy
- Confusion Matrix
- Precision
- Recall

Best Model:
Logistic Regression

Files Included:
- Iris_Classification_Assignment.ipynb
- iris_model.pkl
- scaler.pkl

Inference Example:
Input: [6.1, 2.8, 4.7, 1.2]
Output: Iris-versicolor
